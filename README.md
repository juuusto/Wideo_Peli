To run this project you are going to need these specific .dll files:  
    libatomic-1.dll  
    libgcc_s_seh-1.dll  
    libgfortran-4.dll  
    libgomp-1.dll  
    libquadmath-0.dll  
    libssp-0.dll  
    libstdc++-6.dll  
    libwinpthread-1.dll  
    openal32.dll  
    sfml-audio-2.dll  
    sfml-audio-d-2.dll  
    sfml-graphics-2.dll  
    sfml-graphics-d-2.dll  
    sfml-network-2.dll  
    sfml-network-d-2.dll  
    sfml-system-2.dll  
    sfml-system-d-2.dll  
    sfml-window-2.dll  
    sfml-window-d-2.dll  
These files should be located under src directory.
